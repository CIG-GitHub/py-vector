# Code of Conduct

This project aims to be a constructive, technical space.

## Expected Behavior

- Be respectful and assume good faith.
- Discuss ideas and implementation details, not people.
- Provide criticism that is specific and technically grounded.
- Accept that this project is opinionated and may not suit every workflow.

## Unacceptable Behavior

- Personal attacks, insults, or harassment.
- Repeated misrepresentation of the project after correction.
- Derailing technical discussions into bad-faith arguments.
- Hostile or inflammatory language.

## Enforcement

The project maintainer may remove comments, issues, or contributions that violate these guidelines.

This includes closing issues or discussions that are unproductive, antagonistic, or off-topic.

## Scope

These guidelines apply to all project spaces, including issues, pull requests, and discussions.

Participation in this project implies agreement with this Code of Conduct.
